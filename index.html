About This Calculator
Technical background, formulas, and references.
Overview

This calculator implements the sample size determination methods for Confirmatory Factor Analysis (CFA) and Structural Equation Modelling (SEM) based on the formulas derived by Kim (2005), as described in:

Arifin WN. A web-based sample size calculator for structural equation modelling. Education in Medicine Journal. 2025;17(1):195–212.

Core Formulas

Degrees of Freedom

df = a − b
a = p(p + 1) / 2
b = FL + VAR + COVAR + REG

Baseline df: only item variances freely estimated

Non-Centrality Parameter (NCP)

Computed iteratively given α, power, and df using the chi-squared distribution (Kim, 2005 algorithm).

Sample Size by RMSEA

N = (NCP / (RMSEA² × df)) + 1

Model-Implied Correlation Matrix

Σ = Λy · Ψ · Λyᵀ + Θε

Where Λy = factor loading matrix, Ψ = factor correlation matrix, Θε = diagonal unique variance matrix.

Sample Size by CFI

F_B = −ln(det(Σ))
N = (NCP + df_baseline × (1 − CFI)) / (F_B × (1 − CFI)) + 1

Dropout Adjustment

n_drop = ceil(n / (1 − dropout_rate))
Available Modules
  • CFA by RMSEA — requires items, factors, RMSEA, α, power
  • CFA by CFI (Equal) — requires equal items per factor, CFI, factor loading & correlation
  • CFA by CFI (Advanced) — handles unequal items per factor
  • SEM by RMSEA (General) — requires model df directly
  • Degrees of Freedom — utility to compute model/baseline df
